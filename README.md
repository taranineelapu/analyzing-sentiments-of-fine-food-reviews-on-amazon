# Analyzing Sentiments of Fine Food Reviews on Amazon
This project analyzes customer sentiments from Amazon Fine Food using both VADER Sentiment Analysis and a Logistic Regression model. By comparing a rule-based NLP approach (VADER) with a machine learning approach (Logistic Regression), the project provides insights into customer satisfaction and dissatisfaction, helping product owners better understand feedback for product improvements.

# Objective
* Analyze customer sentiments in Amazon Fine Food Reviews to identify patterns of satisfaction and dissatisfaction.
* Apply VADER Sentiment Analysis and a Logistic Regression model to compare rule-based and machine learning approaches.
* Evaluate the effectiveness of each method in capturing sentiment polarity relative to customer ratings.
* Provide insights that can support product improvement decisions and enhance customer experience.

# Methodology
* **Data Preprocessing:** Cleaned and prepared the Amazon Fine Food Reviews dataset (available on Kaggle) for analysis.
* **Exploratory Analysis:** Conducted EDA to examine distributions of ratings and textual sentiment patterns.
* **Sentiment Modeling:** Applied VADER for rule-based sentiment scoring and trained a Logistic Regression model for classification.
* **Evaluation:** Compared model outputs against customer ratings to assess accuracy and effectiveness.
* **Visualization:** Used matplotlib and pandas plots to illustrate sentiment trends and correlations with ratings.

# Tools & Technologies
**Language:** Python
**Libraries:** NLTK (VADER), scikit-learn, pandas, matplotlib
**Models:** VADER Model, Logistic Regression
